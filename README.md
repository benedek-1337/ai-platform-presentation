# 1337 AI — Platform presentation

A single-file, scroll-driven product presentation for **1337 AI** — the enterprise
intelligence layer that turns a company's systems, data and processes into
AI-ready intelligence.

- **Self-contained** — one HTML file, no build step, works offline (fonts, the React
  runtime and all assets are bundled inline).
- **Bilingual** — English / Hungarian (toggle top-right).
- **Scroll-driven narrative** — hero → the pieces → one framework → the intelligence
  layer → what runs inside the platform → adoption → proof → book a demo.

## View it

Open `index.html` in any modern browser, or enable **GitHub Pages**
(Settings → Pages → deploy from `main`) to host it at a public URL.
